## Running locally

No build step. Pure ES modules.

```bash
git clone https://github.com/seb-iche/pixelium
cd pixelium
npx serve .
```

Or use VS Code Live Server — right-click `index.html` → Open with Live Server.

---

## Deploy to Vercel

```bash
npm i -g vercel
vercel
```

Or connect the GitHub repo at [vercel.com](https://vercel.com) — no build command needed, deploy as static site.
